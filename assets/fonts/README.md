# Fonts

Folder ini untuk menyimpan font custom jika diperlukan.

Font yang direkomendasikan:
- Roboto (default Android)
- Open Sans
- Montserrat

Untuk menggunakan font custom:
1. Letakkan file font (.ttf atau .otf) di folder ini
2. Atur di app_ui.py pada class Fonts:
   FAMILY = 'assets/fonts/NamaFont.ttf'
